# Customer_segmentation_analysis
Customer Segmentation using K-Means  This project uses K-Means clustering to segment customers based on Annual Income and Spending Score. The code loads a dataset, applies clustering to group similar customers, and visualizes the clusters using a scatter plot. , helping identify patterns such as high-spending or low-spending customer groups. 

## Features
- Data preprocessing & feature selection
- K-Means clustering
- Scatter plot visualization
- Business insights based on cluster behavior

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-Learn
- Matplotlib
- Streamlit (for deployment)
